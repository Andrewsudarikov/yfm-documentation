# Заметки

Заметка — выделяющийся блок с важной информацией.

В зависимости от содержания используют следующие типы заметок:
* «Примечание» — дополнительная информация.
* «Совет» — рекомендация.
* «Важно» — предостережение.
* «Внимание» — ограничение.


## Примечание
  ```markdown
  {% note info %}

  Это примечание.

  {% endnote %}
  ```
